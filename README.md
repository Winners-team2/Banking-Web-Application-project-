# 🏦 Banking Web Application Project

A web-based banking system built for educational purposes, combining **web development** and **cybersecurity** concepts.

## Overview
This project simulates a real-world online banking system with intentionally implemented vulnerabilities. 
The goal is to understand how common web security flaws work, how to detect them using security tools, 
and how to properly fix them.

## Purpose
- 🔍 Identify common web application vulnerabilities
- 🛠️ Use security tools to discover weaknesses
- 🔒 Learn how to patch and secure web applications
